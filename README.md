# 🏨 Hotel Booking System

A full-stack Hotel Booking Web Application built using **Node.js**, **Express.js**, **EJS**, and **SQL Database**.

🚧 This project is currently under development.

---

## 📌 Tech Stack

### 🔹 Backend
- Node.js
- Express.js

### 🔹 Frontend
- EJS (Embedded JavaScript Templates)
- HTML5
- CSS3

### 🔹 Database
- MySQL (SQL Database)
